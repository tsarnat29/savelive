- Якщо змінюється вхідна форма з заявкою.

Зробити зміни в програмах, які розташовані в папках client та server:
client:
Внести зміни в програму CreateOrder.js
Внести зміни в програму CreateLogsOrder.js
server:
Внести зміни в програму models.js
Внести зміни в структуру даних в програмах:
-orderController.js
-basketController.js
-logController.js

Зайти в базу даних
Вибрати savelive->Shemas->public->Tables->baskets->Columns
В Columns видалити стовпчик unitId
Вибрати таблицю demands та видалити в неї стовпчик basketId
Тепер, коли залежностей між таблицями не має, можна видалити
таблицю demands (це потреби, які зберігаються окремо для таблиці baskets)
і таблицю baskets.
Також потрібно видалити таблицю logsBaskets, тому ще структура таблиці логів при змінах в формі
теж має змінитися.

Якщо потрібно зберегти ці таблиці, потрібно написати окрему програму, яка змінює
структуру таблиць. (Додає нові стовбчики або видаляє старі)

Після видалення таблиць в терміналі VS Code запустити
сервер: перейти в папку server, якщо ви ще не в цій папці, за допомогою команди cd server. 

Перезапустити сервер, спочатку закривши  (Ctrl+C),
відповісти: y

Далі набрати команду для запуска:
 npm run dev

При запуску таблиці будуть створені згідно з новою структурою.

Далі відкрити ще один термінал, де запустити реакт:
cd client
npm start
